Client Intake Analytics Dashboard (Power BI)

Problem:
Lack of visibility into client intake volume across regions and channels.

Solution:
Built an interactive Power BI dashboard enabling continent-to-country drill-down,
Top-10 country analysis, and channel performance comparison.

KPIs:
- Total Client Requests
- Client Requests by Channel
- Top Countries by Volume

Tools:
Power BI, DAX, Data Modeling, Data Visualization
